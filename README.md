# custom-koishipro-updater


`config.yaml` contains the configuration

- `ignore` -  contains paths you want to ignore
- `path` -  where you want it to download
- `url` - link to the kpro index page

example:
```yaml
ignore:
  - cmake/
  - CMakeLists.txt

path: koishipro
url: https://koishi.pro/ygopro/
```
